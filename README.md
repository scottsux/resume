# resume
This repository contains a project to convert a resume into an Apple-style portfolio website. The project includes three main files: `index.html`, `style.css`, and `script.js`. The website is designed to be minimal, premium, and responsive, following the aesthetic principles of Apple.

这个项目的目标是将简历转换成一个苹果风格的个人作品集网站。该项目包含三个主要文件：`index.html`、`style.css` 和 `script.js`。该网站设计简约、优雅且响应式，遵循苹果的美学原则。
